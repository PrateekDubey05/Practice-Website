HTML5 Elements

Header Tag

Represents introductory content or a set of navigational links for a documnet or section.

Nav Tag

Defines a section containing navigation links

Article Tag

Represents a self-contained composition in a document, such as a blog post, forum post, or news article.

Section tag

Represents a generic standalone section of a document, which doesn't have a more specific semantic Element represent it.

Aside Tag

Represents content taht is tangentially related to the content arount it, often presented as a sidebar

Details Tag

Used together to create an interactive widget that the user can open and close

Summary Tag

Provides a visible heading for the details

Mark Tag

Represents Text which is marked or highlighted for refrence or notation purposes.

Time Tag

Represents a Specific period in time or a date.

Footer Tag

Represents a footer for its nearest sectioning content or the root element. It typically contains information about the author, copyright data, or links to related documents.
